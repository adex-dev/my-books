<summary><strong>Kebijakan Penggunaan Repository</strong></summary>

Repository ini dibuat untuk kebutuhan dokumentasi internal, pembelajaran, pengembangan sistem, dan operasional tertentu.

Dengan mengakses atau menggunakan repository ini, pengguna dianggap memahami dan menyetujui seluruh kebijakan berikut:

### Aturan Penggunaan

- Dilarang menyebarkan isi repository tanpa izin pemilik.
- Dilarang mengunggah ulang sebagian atau seluruh isi repository ke platform lain.
- Dilarang mengklaim dokumentasi, source code, module, atau asset sebagai milik pribadi.
- Dilarang menggunakan isi repository untuk aktivitas ilegal atau merugikan pihak lain.
- Dilarang memperjualbelikan isi repository tanpa izin tertulis dari pemilik.

### Akses dan Distribusi

Pemilik repository berhak:

- Membatasi akses pengguna
- Mengubah isi repository kapan saja
- Menghapus atau memindahkan dokumentasi tanpa pemberitahuan
- Menolak permintaan distribusi ulang

### Tanggung Jawab Pengguna

Pengguna bertanggung jawab penuh atas penggunaan dokumentasi, source code, firmware, module, maupun asset yang terdapat pada repository ini.

Segala risiko yang timbul akibat penggunaan repository menjadi tanggung jawab pengguna masing-masing.

### Kebijakan Perubahan

Kebijakan ini dapat diperbarui sewaktu-waktu tanpa pemberitahuan sebelumnya.

Dengan tetap menggunakan repository ini, pengguna dianggap menyetujui versi kebijakan terbaru.
