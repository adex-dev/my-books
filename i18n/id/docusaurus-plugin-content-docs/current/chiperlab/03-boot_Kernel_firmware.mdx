## Boot

Boot adalah metode menjalankan perangkat ke mode download khusus untuk proses flashing firmware dan recovery sistem. Mode ini digunakan ketika perangkat mengalami bootloop, stuck logo, atau gagal masuk sistem normal.

Dengan mode MD, perangkat dapat terhubung ke komputer agar firmware dapat di-install ulang secara langsung ke memori perangkat.

### cara instalasinya

Pastikan anda sudah mendownload tools yang diperlukan pada artikel flash.

### Boot Installation Flow

```mermaid id="q7xa9m"
flowchart TD

A[Turn Off Device] --> B["Press 1 + 7 + Power"]
B --> C[Select Load Program]
C --> D[Choose Connection Type]
D --> E[Select Cradle-IR]
E --> F[Set Baud Rate 115200]
F --> G[Insert Device into USB Docking]
G --> H[Open ProgLoad.exe]
H --> I[Load B8000V423.shx]
I --> J[Click OK]
J --> K[Wait Until Download Complete]
````

1. Matikan dan nyalahkan kembali device dengan menekan kombinasi :

   ```text
   1 + 7 + Power
   ```

2. Pilih **Load Program**

3. Pilih jenis koneksi jaringan:
  
  ```text
  IrDA atau Cradle-IR
  ```

4. Saya pilih:

  ```text 
  Cradle-IR 
  ```

5. Pilih baud rate :

    ``` text 
    115200
    ```

6. Masukan Perangkat Chiperlab U-8000 kedalam USB docking.

7. Buka Software :

   ```text
   ProgLoad.exe
   ```

  <Admonition type="info">
Software ini terdapat pada folder beserta Firmware, Boot, and Kernel Package yang telah di download sebelumnya.
  </Admonition>

8. Pilihan Settingan `ProgLoad.exe`

  * **Command Type** → `Cradle-IR`

  * **Com Port** → sesuaikan dengan device manager

  * **Baud Rate** → `115200 bps` atau sesuai perangkat

  * **File Type** → `.SHX`

  * **Browse** → mencari file `boot.shx`

  * **Klik OK** 

9. Tunggu hingga Progress Download Selesai

## Kernel & Firmware

masih di program:

 ```text
   ProgLoad.exe
   ```

### Pilihan Settingan ProgLoad.exe

  * **Command Type** → `Cradle-IR`
  * **Com Port** → sesuaikan dengan device manager
  * **Baud Rate** → `115200 bps` atau sesuai perangkat
  * **File Type** → `.SHX`

  ### Installation Flow

```mermaid id="h9el2u"
flowchart TD

A[Open ProgLoad.exe] --> B[Browse K8000V334.shx]
B --> C[Click OK]
C --> D[Wait Until Complete]
D --> E[Browse U-8000xxx.shx]
E --> F[Click OK]
F --> G[Wait Until Complete]
G --> H[Restart Device]
H --> I[Press ESC to Main Screen]
```

  * **Browse** → mencari file `K8000V334.shx`
  * **OK** → Tunggu setelah selesai ulangi
  * **Browse** → mencari file `U-8000xxx.shx`
  * **OK** → Tunggu hingga selesai dan matikan perangkat lalu hidupkan kembali
  * **ESC** -> tekan `esc` untuk ke layar utama.



  ### Catatan
  <Admonition type="warning">
  > pastikan perangkat terhubung dengan docking dengan stabil. sebelum klik OK, pastikan klik Load Program pada perangkat chiperlab. Jangan terlalu lama, karena mempunyai batas waktu permintaan koneksi, sekitar 5 - 10detik. jika lebih maka ulangi kembali.
  </Admonition>

Selanjutnya kita akan masuk ke instalasi modul.
