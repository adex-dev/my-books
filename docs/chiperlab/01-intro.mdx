# Intro Chiperlab

## Chiperlab Version 8000

This documentation uses the Chiperlab 8000 device type.

The Chiperlab 8000 is commonly used for warehouse auditing activities, including inventory validation, stock checking, and barcode scanning operations. This device helps improve operational accuracy and efficiency during warehouse management processes.
